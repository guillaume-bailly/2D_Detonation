## Installation

```bash
git clone https://github.com/guillaume-bailly/2D_Detonation.git
cd 2D_Detonation
pip install -r requirements.txt
```
Or:

```bash
pip install 2d_detonation
```

And update with:

```bash
pip install --upgrade 2d_detonation
```
